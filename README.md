# System-Administration-Tool


## Features:
- [x] Get and Upload custom commands for files
- [x] All Terminal (win/linux) commands
- [x] Command logs file

### Usage:
First start admin.py with command

For linux:
`python3 admin.py`

For windows:
`python -m admin.py`

-------------------
And then client.py

For linux:
`python3 client.py`
For windows:
`python -m client.py`
